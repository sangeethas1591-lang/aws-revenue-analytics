REVENUE PERFORMANCE ANALYSIS


📌 Project Overview


End-to-end revenue analytics project for a retail company aimed at improving business decision-making through customer pattern recognition and regional sales analysis.

🛠️ Technical Stack
* Cloud Infrastructure: AWS (S3, Glue, Lake Formation, Athena, CloudWatch)
* Data Science: Python (Data cleaning, Hypothesis testing, Regression modeling)
* Visualization: Power BI (Interactive Dashboards)

🏗️ Data Architecture
1. Extract & Store: Raw data ingested into Amazon S3.
2. Catalog & Transform: AWS Glue crawlers and ETL jobs to build the data catalog.
3. Govern: AWS Lake Formation for permission management and security.
4. Query: Amazon Athena for serverless SQL querying.

📈 Key Insights & Recommendations
* Top Category: Electronics ($24k revenue leader).
* Forecasting: Built a 2025 Quantity Forecast peaking in Q1.
* Strategy: Recommended targeted marketing for "Fashion" and regional expansion
